The URF517 Reflector has been upgraded with a change to the functionality of YSF developed by Tom Early, N7TAE.

This change allows the operator to switch between channels/modules using DG-ID.

Reflectors can be enabled with up to 26 channels, A through Z.  For YSF, each channel is assigned a DG-ID beginning with "10" for channel "A," "11" for channel "B," "12" for "C" and so forth.

URF517 currently has eight channels enabled, A thru E and X thru Z.

DMR has a similar feature.  Since DMR and YSF use the same codec DMR and YSF can have QSOs even when not on channel "A," the main transcoded channel.

Enjoy and let us know if you should have any questions.
